## About

A Unity prototype exploring action-game mechanics built around an unusual constraint: the player has no weapon. Instead of the typical melee loop, the design centers on **improvisation** — using the environment, momentum, and timing to survive.

Built as a personal exploration of player movement, combat feel, and how much expressiveness you can squeeze out of a tightly constrained kit.
